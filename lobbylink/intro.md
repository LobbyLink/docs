# Introduction

**"I just want to use WebRTC. I dont want to reinvent the wheel and gert startet as quickly as possible."** This is no problem. For this exact reason I created LobbyLink. It is a library that lets you initiallize a WebRTC connection with only 3 commands.


## Roadmap

In the future I plan to extend it to more protocolls and have one big monorepo where you can cherry-pick multiplayer features that improve you Qualityof Life while developing. But these are dreams of the future.
